# LFDD

This repository presents "Landmark-based feature drift detector" proposed by Jean Paul Barddal et al.[(A survey on feature drift adaptation)](https://doi.org/10.1016/j.jss.2016.07.005). 

## Getting Started

### Prerequisites

* JDK 7+
* MOA 2016+
* Weka 3.6.7+
